Approven van pull requests
ga naar https://github.com/Alwin1234567/Modelleren/pulls
kijk of er nieuwe pull requests zijn
Als die er zijn
kijk er naar door op files changed te drukken en check of het goed is.
	Voor uitgebreidere check il je waarschijnlijk de branch lokaal bekijken op je laptop

Als die goed is kan je rechts boven bij files changed op review changes drukken. Daarmee kan je commentaar toevoegen als het nodig is en het approven als het goed is


rebasen na pull request

ga naar de branch die ge rebased meot worden in source tree
druk op het terminal logo

doe "git rebase -i <naam van branch waarnaartoe gerebased oet worden>"
Je krijgt een lijst met commits, type ":wq" en druk op enter
nu is die lokaal gerebased.
Nu moet je het naar origin pushen, dit doe je door 
"git push --force-with-lease"
te typen
